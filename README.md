# 💪 MuscleMind AI

MuscleMind AI is an AI-powered fitness web application that helps users generate personalized workout plans, diet plans, and fitness guidance using Artificial Intelligence.

Built for the **Elite Coders Open Source Hackathon**.

---

## 🚀 Features

### 🏋️ AI Workout Generator

Generate personalized workout routines based on:

* Weight
* Height
* Goal
* Experience level
* Workout days

### 🥗 AI Diet Planner

Get AI-generated diet plans based on:

* Goal (Muscle Gain / Fat Loss)
* Food preference
* Budget
* Meals per day

### 💬 AI Fitness Coach

Ask fitness-related questions like:

* Workout advice
* Diet guidance
* Recovery tips
* Gym mistakes
* Protein intake

### 📈 Progress Tracking

Track:

* Weight changes
* Waist size
* Fitness notes

Includes **interactive charts** for:

* Weight Progress
* Waist Progress

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/kunukumikhil-byte/MuscleMind-AI.git
cd MuscleMind-AI
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Create `.env`

```env
GROQ_API_KEY=your_api_key_here
SECRET_KEY=musclemind_secret
```

### Run Application

```bash
python app.py
```

Open:

```txt
http://127.0.0.1:5000
```
